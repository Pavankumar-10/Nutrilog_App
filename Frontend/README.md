# 🥗 NutriLog – Nutrition Tracking System

## 📌 Overview

NutriLog is a full-stack nutrition tracking application that helps users monitor their daily food intake and nutritional values.

The application allows users to register, log in securely, add food entries, view nutrition history, and track their progress through an interactive dashboard.

---

## ✨ Features

- User Registration
- Secure Login with JWT Authentication
- Dashboard
- Add Food Entry
- Food History
- Nutrition Calculation
- Responsive User Interface

---

## 🚀 Installation

### Backend

1. Open Backend in Eclipse or STS.
2. Configure MySQL.
3. Run the Spring Boot application.
